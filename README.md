# PoE Timeless Jewels

流亡黯道（Path of Exile）永恆珠寶查詢與計算工具。

## 主要功能

- 依珠寶類型、種子（Seed）與條件篩選結果
- 顯示對應詞綴與天賦樹影響
- 支援 JSON 匯出與匯入
- 前端以 WASM 計算，提供快速查詢體驗
